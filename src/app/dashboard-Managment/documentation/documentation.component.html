Frequently Asked Questions

Why should I use ViewVoice when I can directly use Azure Document Intelligence?

- You should use ViewVoice because it enhances Azure Document Intelligence by offering a user-friendly interface tailored for expense analysis.
- While Azure Document Intelligence focuses on extracting data from documents, ViewVoice takes this further by transforming the data into interactive visual reports and providing insights that help you manage your expenses more effectively.
- ViewVoice simplifies the complex JSON data and allows you to make sense of it quickly, without needing technical expertise in handling raw output from Azure DI.

How do I use ViewVoice for analysing multiple invoices?

Step 1 - Go to the PDF Uploader panel and simply choose the files you want analyse (make sure they are in PDF format).
Step 2 - Navigate to the Insights panel or click on the View Insights button to instantly view all the charts and the table created from your invoivces.

What are the different ways I can export data from ViewVoice?

ViewVoice provides varois ways to export the data fetched from your invoices.
- The table can be exported to Excel for custom modification.
- All the charts relevant to your invoices can be exported to a single PDF.
- Most charts can be individually exported to SVG, PNG as well as CSV formats.
